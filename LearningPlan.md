# Project Goal

My final project is called FakeBook, a Facebook-like web application.

The project will include:
- Users
- Posts
- Comments

By the end of Week 5, I want to build a working React frontend that connects to my REST API backend for FakeBook.

The frontend should support:
- User login
- Authentication
- Viewing posts
- Creating posts
- Viewing comments
- Sending authenticated API requests
- Basic responsive design

I plan to spend around 6–10 hours per week learning and practicing React.
